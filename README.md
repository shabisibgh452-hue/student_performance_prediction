# Student Performance Prediction

## Objective
Predict student performance using historical academic data.

## Models Used
- Logistic Regression
- Random Forest

## Technologies
- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit
- Matplotlib
- Seaborn

## How to Run
1. Install libraries:
   pip install -r requirements.txt

2. Train the model:
   python training.py

3. Run the Streamlit app:
   streamlit run app.py
